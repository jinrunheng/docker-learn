## 容器技术与 Docker 简介
Vagrant
VirtualBox

Vagrant 的基本使用
```bash
vagrant version
```
使用 vagrant 快速创建虚拟机
```bash
vagrant init centos/7x
```

启动虚拟机
```bash
vagrant up
```
通过 ssh 链接进入虚拟机
```bash
vagrant ssh
```

查看当前运行的虚拟机
```bash
vagrant status
```
停止虚拟机
```bash
vagrant halt
```
删除虚拟机
```bash
vagrant destroy
```
